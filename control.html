Control set

When thinking about reaching the target area from particular location it can be useful to think about the control set: a set of points, where you can reach the target from.

It is important to understand how the shape of this set depends on wind speed and direction. In the following demonstration try changing the wind speed and direction. The blue area on the map shows which points you can reach the target area from if you're at altitude of 700m
